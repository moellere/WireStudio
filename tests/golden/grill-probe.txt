+--------------------------------------------------------------------------+
|  ESP32-DevKitC-V4  ---  grill-probe                                      |
+--------------------------------------------------------------------------+
|  Rails: 5V, 3V3, GND                                                     |
|                                                                          |
|  probe  [MAX31855 thermocouple amplifier (SPI, K-type)]  -- Grill probe  |
|    VCC   -> rail 3V3                                                     |
|    GND   -> rail GND                                                     |
|    CLK   -> spi0 (GPIO18)                                                |
|    MISO  -> spi0 (GPIO19)                                                |
|    CS    -> GPIO5                                                        |
|                                                                          |
|  lid  [MPU6050 6-axis IMU (accel + gyro, I2C)]  -- Grill lid             |
|    VCC   -> rail 3V3                                                     |
|    GND   -> rail GND                                                     |
|    SDA   -> i2c0 (GPIO21)                                                |
|    SCL   -> i2c0 (GPIO22)                                                |
|                                                                          |
|  Passives:                                                               |
|    c1: 100nF capacitor, probe.VCC  <->  GND   (decoupling probe)         |
|    c2: 100nF capacitor, lid.VCC  <->  GND   (decoupling lid)             |
|    r1: 4.7k resistor, i2c0.SDA  <->  3V3   (I2C pull-up)                 |
|    r2: 4.7k resistor, i2c0.SCL  <->  3V3   (I2C pull-up)                 |
|                                                                          |
|  BOM:                                                                    |
|    - ESP32-DevKitC-V4                                                    |
|    - MAX31855 thermocouple amplifier (SPI, K-type)  (probe)              |
|    - MPU6050 6-axis IMU (accel + gyro, I2C)  (lid)                       |
|    - 2x 100nF capacitor                                                  |
|    - 2x 4.7k resistor                                                    |
|                                                                          |
|  Power: ~135mA typical, ~508mA peak (budget 700mA)  OK                   |
|                                                                          |
|  Warnings: none                                                          |
+--------------------------------------------------------------------------+
