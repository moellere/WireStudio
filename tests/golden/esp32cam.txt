|    D4    -> GPIO36                                                                                             |
|    D5    -> GPIO39                                                                                             |
|    D6    -> GPIO34                                                                                             |
|    D7    -> GPIO35                                                                                             |
|    VSYNC -> GPIO25                                                                                             |
|    HREF  -> GPIO23                                                                                             |
|    PCLK  -> GPIO22                                                                                             |
|    PWDN  -> GPIO32                                                                                             |
|                                                                                                                |
|  BOM:                                                                                                          |
|    - AI-Thinker ESP32-CAM                                                                                      |
|    - ESP32 OV2640 / OV7670 / OV5640 camera  (cam)                                                              |

|                                it low only for flashing.                                                       |
