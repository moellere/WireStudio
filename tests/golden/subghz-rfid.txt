|                                                                                 |
|  rfid  [RDM6300 125kHz EM4100 RFID reader]  -- RFID reader                      |
|    VCC   -> rail 5V                                                             |
|    GND   -> rail GND                                                            |
|    TX    -> rfid_uart (GPIO16)                                                  |
|                                                                                 |
|  Passives:                                                                      |
|    c1: 100nF capacitor, radio.VCC  <->  GND   (decoupling radio)                |

|  BOM:                                                                           |

|    - Texas Instruments CC1101 sub-GHz transceiver (SPI)  (radio)                |

|    - 1x 100nF capacitor                                                         |
