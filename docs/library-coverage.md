# Library coverage

Generated by `scripts/coverage_matrix.py`. Each entry's checkbox flips based on whether at least one bundled example in `wirestudio/examples/` uses it. Bundled examples are validated against pinned ESPHome by the [esphome-config gate](../.github/workflows/esphome-config.yml), so a ✅ here means that library entry round-trips end-to-end through real ESPHome -- not just that the YAML template renders. Radio (LoRaWAN) boards are exercised instead by the lorawan firmware build.

**Summary:** components 60/60 (100%), boards 23/23 (100%).
